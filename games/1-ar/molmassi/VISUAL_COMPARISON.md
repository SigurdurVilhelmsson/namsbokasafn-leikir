## Interaction Comparison

### OLD Game Loop:
```
1. See formula (text)
2. Open periodic table
3. Find atomic masses
4. Do math on paper/calculator
5. Type number
6. Click submit
7. Hope for green checkmark
8. Time: 2-3 minutes
9. Understanding: "I memorized the process"
```

### NEW Game Loop:
```
1. See visual molecule
2. Engage with challenge (count/compare/build)
3. Make selection
4. Get immediate feedback
5. Read educational note
6. Move to next challenge
7. Time: 20-40 seconds
8. Understanding: "I know what molecules are made of"
```

## Why This Matters

### Cognitive Load Theory
- Visual < Symbolic < Abstract
- Pictures are processed faster than text
- Spatial reasoning builds mathematical intuition

### Constructivism
- Students build knowledge through interaction
- Active manipulation > passive observation
- Mistakes are learning opportunities

### PhET Principles
- "Manipulate, observe, explain"
- Immediate feedback
- Multiple representations
- Low floor, high ceiling
