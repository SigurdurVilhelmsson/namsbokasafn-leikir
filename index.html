About Chemistry Games

Chemistry Games is a collection of interactive educational games designed to help first-year chemistry students master fundamental concepts through engaging, hands-on practice. Each game focuses on different aspects of chemistry education, from nomenclature to stoichiometry.

Built with modern web technologies including React and Tailwind CSS, these games provide an intuitive, responsive learning experience that works on any device.
